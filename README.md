## Getting Started

To begin:

1. **Clone the repository**
    ```bash
    git clone https://github.com/Muhammad-Roshaan-Idrees/Operating-System.git
    ```
2. **Navigate to the project directory**
    ```bash
    cd Operating-System
    ```
3. **Recommended Environment**
    - Ubuntu 20.04 or newer
